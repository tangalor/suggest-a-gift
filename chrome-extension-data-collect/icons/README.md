# Icone per l'estensione

Inserisci qui le icone dell'estensione:

- **icon16.png** - 16x16px (per la toolbar)
- **icon48.png** - 48x48px (per la pagina delle estensioni)
- **icon128.png** - 128x128px (per il Chrome Web Store)

## Suggerimenti per le icone

Le icone dovrebbero rappresentare il tema "Data Collect" con i colori del brand:
- Verde: #2d5016 o #1a3a0f
- Oro: #ffd700

Puoi usare strumenti come:
- Figma, Adobe Illustrator, Canva
- Generatori di icone online
- AI image generators

Per ora l'estensione funzionerà anche senza icone personalizzate (Chrome userà un'icona predefinita).
